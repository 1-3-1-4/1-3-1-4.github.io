Python--07-逻辑条件

发表于 2021-04-30 更新于 2021-05-07 分类于 Python 阅读次数：
基本的流程控制、条件判断

一、条件控制

1、if语句

if 条件:
    程序代码
程序代码

示例：

var1 = True
if var1:
  print('真区间')
print('结束')

2、if ... elif ...语句

示例：

numEx = 10
if numEx > 0 && numEx < 10:
  print('大于0小于10')
elif numEx >= 10 && numEx < 20:
  print('大于等于10小于20')
else:
  print('其他')
  
# 大于等于10小于20

二、循环

1、while循环

while 判断条件:
  执行语句

示例：

intEx = 1
while intEx < 10:
  intEx += 2
print(intEx)
# 11

## 循环条件一直为真
intEx = 1
intEx2 = 0
while intEx == 1:
  if intEx2 > 5:
    break
  intEx2 += 1
print(intEx2)
# 6

2、 while ... else ...语句

在条件语句为false时执行else的语句块

intEx = 6
while intEx > 5:
    intEx -= 1
    print('intEx 大于 5')
else:
    print('intEx 不大于 5')
# intEx 大于 5
# intEx 不大于 5

3、 for语句

for <variable> in <sequence>:
    <statements>
else:
    <statements>

示例：

listEx = ['python', 'php', 'golang']
for item in listEx:
  print(item)
else:
  print('循环结束')

4、 break 和 continue

break 是跳出循环

continue是结束此次循环进入下次循环

5、 pass

是空语句，不做任何事情，一般用做站位语句
